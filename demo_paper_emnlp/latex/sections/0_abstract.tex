\begin{abstract}
We present InfiniSST, a production-ready multi-user speech-to-speech translation system that achieves high throughput and low latency through novel ORCA-inspired scheduling. Our system addresses the critical challenge of serving multiple concurrent users for real-time speech translation while maintaining sub-second latency. InfiniSST leverages a sophisticated architecture that separates prefill and decode operations, implements dynamic batching with attention sink mechanisms, and efficiently manages GPU resources to support up to 32 concurrent users on a single GPU. The system features comprehensive functionality including dynamic language pair switching, multiple latency options, system audio capture, and both web and desktop applications. Through careful optimization of the inference pipeline and innovative scheduling strategies, InfiniSST achieves an average latency of 387ms with 99.2\% success rate, representing a significant advancement in production SST systems.
\end{abstract}
