\section{Limitations and Conclusion}
\label{sec:limitations}

The main evaluation is En$\rightarrow$Zh and uses ten registered topic slices.
Those slices contain the benchmark terminology, so the results measure routing
and retrieval competition rather than unseen-glossary induction. Routing also
depends on generated target context and can lag after a topic boundary; the
multi-slice budget only reduces this failure mode. The degradation threshold
also depends on the language pair, retriever, ambiguity, and prompt budget;
100k is an intermediate point and 1M a stress condition, not a universal
threshold. Fresh multilingual evaluation remains future work.

AutoTerm-SST turns terminology retrieval into a zero-session-setup system.
Operators expand a modular offline catalog while users simply start a stream;
recent context maintains a compact working set and RASST supplies prompt
evidence. On mixed ACL and medicine talks, AutoTerm matches a known-domain
reference, improves over no RAG, and avoids the quality and prompt-noise
degradation of a 1M universal glossary. The demo makes this behavior visible
and provides a practical path across many domains.
